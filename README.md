# HI-LOW — ハイロー（飲み版）

スマホ1台あれば回しながら遊べる、プッシュ・ユア・ラック系の飲みゲーム。

## 遊び方

1. プレイヤー名を入力（2〜8人）→ START
2. 自分の番が来たら「タップで開始」
3. 表示されたカードの **次のカード** が **HI（大きい）** か **LO（小さい）** か予想
4. 当たったら連続成功 +1 → 続けるか **SAFE** で降りるか選ぶ
5. 外したら 「外したら ○杯」 を一気飲み → 次の人へ

## ルール詳細

- **当たったら** 連続成功カウントが +1、続けて挑戦できる
- **外したら** `連続成功 ÷ 5 + 1`杯を飲む（小数点切り捨て）<br>0〜4回 → 1杯 / 5〜9回 → 2杯 / 10〜14回 → 3杯 …
- **SAFE** 1回でも当てた後ならいつでも降りられる。飲まずに次の人へ
- **同じ数字** が出たら当たり扱い（ドローもセーフ）
- **カード強さ** A=1（最弱） → 2 → … → K=13（最強）

## 動かす

`index.html` をブラウザで開くだけ。スマホで遊ぶときはローカルサーバ越しに：

```bash
cd hi-low
python -m http.server 8000
# 同じWi-Fi内のスマホから http://<PCのIP>:8000 にアクセス
```

外部依存ゼロ。完全オフラインで動く。

## デプロイ

ビルド不要なのでGitリポジトリをそのまま [Cloudflare Workers / Pages](https://workers.cloudflare.com) に繋げば公開できる。

## 技術

- Vanilla JS / 単一HTML / CSS Container Queries（カード自動スケーリング）
- 連続成功 = 賭け杯数 のシンプル機構
- `localStorage` でプレイヤー名の前回入力を保存

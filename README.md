# HI-LOW — ハイロー（飲み版）

スマホで開いてすぐ遊べる、シンプルなカード予想の飲みゲーム。
プレイヤー登録なし、皆で1台を回しながら「次のカード上か下か」当てるだけ。

## 遊び方

1. START を押す
2. 表示されたカードの **次のカード** が **HI（大きい）** か **LO（小さい）** か予想
3. 当たったら連続成功 +1、続けて次のカードを予想
4. 外したら **1杯** 飲む（誰が飲むかは皆で決める）→ 連続成功リセット
5. 同じ数字が出たら **もう一度選ぶ**（連続成功は変わらない）

## ルール

- **当たったら** 連続成功 +1、続けて挑戦
- **外したら** 1杯飲む、連続成功リセット
- **同じ数字** もう一度選ぶ、連続成功は変わらない
- **カード強さ** A=1（最弱） → 2 → … → K=13（最強）
- セッション中の **累計杯数** を表示

## 動かす

`index.html` をブラウザで開くだけ。スマホで遊ぶときはローカルサーバ越しに：

```bash
cd hi-low
python -m http.server 8000
# 同じWi-Fi内のスマホから http://<PCのIP>:8000 にアクセス
```

外部依存ゼロ。完全オフラインで動く。

## デプロイ

ビルド不要。Cloudflare Workers に GitHub リポジトリを繋げば公開できる。

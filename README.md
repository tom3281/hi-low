# HI-LOW — ハイロー（飲み版）

スマホで開いてすぐ遊べる、プッシュ・ユア・ラック系の飲みゲーム。
プレイヤー登録なし、皆で1台を回しながら「次のカード上か下か」当てるだけ。

## 遊び方

1. START を押す
2. 表示されたカードの **次のカード** が **HI（大きい）** か **LO（小さい）** か予想
3. 当たったら連続成功 +1 → 続けるか **SAFE** で連続リセットするか選ぶ
4. 外したら 「外したら ○杯」 を一気飲み（誰が飲むかは皆で決める）→ 連続リセット

## ルール詳細

- **当たったら** 連続成功カウントが +1、続けて挑戦できる
- **外したら** `連続成功 ÷ 5 + 1`杯を飲む（小数点切り捨て）<br>0〜4回 → 1杯 / 5〜9回 → 2杯 / 10〜14回 → 3杯 …
- **SAFE** 1回でも当てた後ならいつでも降りられる。飲まずに連続リセット
- **同じ数字** が出たら当たり扱い（ドローもセーフ）
- **カード強さ** A=1（最弱） → 2 → … → K=13（最強）
- セッション中の **累計杯数 / 最長連続** を表示

## 動かす

`index.html` をブラウザで開くだけ。スマホで遊ぶときはローカルサーバ越しに：

```bash
cd hi-low
python -m http.server 8000
# 同じWi-Fi内のスマホから http://<PCのIP>:8000 にアクセス
```

外部依存ゼロ。完全オフラインで動く。

## デプロイ

ビルド不要。Cloudflare Workers に GitHub リポジトリを繋げば公開できる。

## 技術

- Vanilla JS / 単一HTML / CSS Container Queries（カード自動スケーリング）
- `floor(streak/5) + 1` のステーク式
